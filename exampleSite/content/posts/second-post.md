Hugo is a fast static site generator written in Go. Here's why I chose it and how this theme works.

## Speed

Hugo builds are incredibly fast. Even large sites compile in milliseconds.

## Theming

This Win95 theme wraps everything in window chrome — title bars, buttons, status bars. Every page is a "window" on the desktop.

## Try it yourself

1. Install Hugo
2. Clone this theme
3. Run `hugo server`
4. Enjoy the nostalgia
